Basic Requirements

Basic Requirements for all Qlik applications.

  • Concise app titles and descriptions, example below.

  • Standard icon size (380 x 236)

  • Please show the latest update/reload of data using Date/Time/Time Zone on all visuals, example below.

    • Last Update 07/29/2020 10:00 AM EST

  • Standardize basic data elements

      • Area = Needs to be upper case ex: CAPITAL METRO, EASTERN
      District = Needs to be upper case ex: ATLANTA
      MPOO = Needs to be concatenation of upper case District and upper case MPOO separated by a dash ex: ATLANTA-MPOO2
      ZIP = Needs to be 5 digit ZIP Code ex: 75075
      Route = Needs to be concatenation of 5 digit ZIP Code and upper case last 4 characters of route number separated by a dash ex: 00601-C002

Branding Consistency

  • Use either the Corp. Desktop Theme or the Corp. Mobile Theme on all dashboards, examples below.
