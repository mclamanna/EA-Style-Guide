Forms/Filters

Forms and filters allow users to interact with a page while either entering information or making selections. Filtering is used to eliminate items from the set, creating a smaller and more relevant subset of data.

Bootstrap provides several form control styles, detailed information can be found at the Official Bootstrap Site

Radio Buttons

Radio buttons are for selecting one option from many. On page load one option is selected (always) by default. We use Custom Forms for more cross browser consistency.

Checkboxes

Checkboxes are for selecting one or several options in a list. We continue to use Custom Forms for more cross browser consistency when creating checkboxes.

Select/Dropdown Menu

Select menu (or dropdown) is used when a number of options is greater than 4 (with some exceptions*) and fewer than 15**. Detailed information can be found at the Official Bootstrap Site.

    In filters with a parent-child relationship (when a number of options in one filter determined by a selection made in another filter).
    If more than 15 options, use a text field with autocomplete suggestions.
