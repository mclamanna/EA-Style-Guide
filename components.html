Forms/Filters

Forms and filters allow users to interact with a page while either entering information or making selections. Filtering is used to eliminate items from the set, creating a smaller and more relevant subset of data.

Bootstrap provides several form control styles, detailed information can be found at the Official Bootstrap Site

Radio Buttons

Radio buttons are for selecting one option from many. On page load one option is selected (always) by default. We use Custom Forms for more cross browser consistency.

Interactions (mobile)

Recommended Interactions on mobile devices.

- Detect touch-enable device and turn :hover interactions into click/tap interactions:

  • tooltips appear on first tap;
  • first tooltip disappears when second bar is tapped;
  • tooltip disappears when user taps ANYWHERE else on the screen;
  • second tap on the same bar closes its tooltip and triggers “drill down” function;
  • Icon size (font-size: 14px) - fa-lg <i class="fas fa-undo fa-lg"></i>
  • Interactive target area + padding 5px <a class="reset" style="padding: 5px"></a>
Tools (Font Awesome icons)

Reset – (fa-undo) not visible on landing

Search (fa-search) – visible on landing

Sort (fa-sort) – visible on landing (if needed)
